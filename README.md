# Tic-Tac-Toe

Tic Tac Toe in C

## How to play

Launch the game with by executing the _**game**_ file. Then, enter the coordinators by entering the row and column (in classic format: for row 1 enters 1 etc...).

You can also change the size grid by changing the lenght at the top of the _**main.c**_ file (replace the number after ```#define LENGHT X```) and the game will automatically changes (but you cannot put a grid size under 2). But you have now to complete the entire row or column or diagonal to win !

## End of the game

The game ends when one of the two players wins by aligning his symbol in line, column or diagonal.
